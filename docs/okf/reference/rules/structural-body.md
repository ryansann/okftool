# body/structural-body

Headings, lists, and tables retrieve far better than a wall of prose and give agents anchors to cite.

- **Category:** body
- **Default severity:** warn
- **Fixable:** no

Configured in [.okftool.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
