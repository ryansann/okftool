# no-relative-links

A blanket ban on relative links for bundles that want every cross-link to survive refactors.

- **Category:** linking
- **Default severity:** off
- **Fixable:** yes

Configured in [.okftool.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
