# body/body-not-empty

A frontmatter-only stub is fine to commit, but worth flagging — it carries no prose for retrieval.

- **Category:** body
- **Default severity:** warn
- **Fixable:** no

Configured in [.okftool.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
