# require-description

The description is the unit of progressive disclosure — it is what index snippets and search results show.

- **Category:** frontmatter
- **Default severity:** warn
- **Fixable:** no

Configured in [.okflint.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
