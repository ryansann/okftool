# no-self-link

A self-link adds a noise edge to the graph and conveys nothing.

- **Category:** linking
- **Default severity:** warn
- **Fixable:** no

Configured in [.okftool.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
