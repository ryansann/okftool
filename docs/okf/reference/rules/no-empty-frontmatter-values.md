# no-empty-frontmatter-values

An empty string reads as documented to tooling while conveying nothing; omit the key or fill it in.

- **Category:** frontmatter
- **Default severity:** warn
- **Fixable:** no

Configured in [.okftool.yaml](/reference/configuration.md); see [validation](/reference/validation.md) for the spec layer.
