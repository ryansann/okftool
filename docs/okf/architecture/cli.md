# okflint-cli

The `okflint` binary: `validate`, `lint`, `rules`, `explain`, and `init`.

- Emits pretty, JSON, or SARIF output.
- Exit code is driven by [.okflint.yaml](/reference/configuration.md) `ci.fail-on`.

See the [overview](/architecture/overview.md).
